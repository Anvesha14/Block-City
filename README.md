# 🏙️ BlockCity Hub — Web3 Decentralized Application

BlockCity Hub is a Web3-based decentralized application (dApp) that simulates a digital city ecosystem using blockchain technology. It integrates tokens, NFTs, and DAO governance to demonstrate how decentralized systems work in real-world scenarios.

---

## 🚀 Features

* 🔗 Connect wallet using MetaMask
* 🪙 ERC-20 Token system (Treasury)
* 🪪 NFT-based identity system
* 🏛 DAO governance (proposal & voting system)
* 🌐 Fully decentralized frontend (no backend required)
* ⚙️ Smart contract integration using Ethers.js

---

## 🧠 Project Overview

BlockCity represents a virtual city where users can:

* Manage digital assets (tokens)
* Create identity using NFTs
* Participate in governance through voting
* Interact with blockchain smart contracts

This project demonstrates the working of a complete Web3 ecosystem in a simplified manner.

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Blockchain Interaction:** Ethers.js (v6)
* **Wallet:** MetaMask
* **Smart Contracts:** Solidity
* **Network:** Sepolia Testnet

## ⚙️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Anvesha14/Block-City.git
```

2. Open the project folder

3. Open `index.html` in your browser

4. Connect MetaMask and switch to **Sepolia Testnet**
